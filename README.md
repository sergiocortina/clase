# Compliance Empresarial — Presentación

Presentación interactiva sobre Compliance y puesta en operación de empresas en México. 26 slides con navegación por teclado, timeline, zoom, alto contraste y gráficas animadas.

## Cómo usar

Abrir `index.html` en cualquier navegador.

## Atajos

| Tecla | Acción |
|---|---|
| ← → | Navegar |
| G | Ir a slide |
| F | Fullscreen |
| C | Alto contraste |
| + / - | Zoom |
| ? | Ayuda |

Fuentes: COFEMER, CONAMER, SAT, IMSS, LGSM.
